# 📈 Reliance Stock Analysis (2000 - 2021)

A complete Data Science project analyzing 21 years of Reliance Industries 
stock market data using Python.

## 🔍 What this project does
- Loads and cleans 21 years of real stock market data
- Explores patterns and trends using EDA
- Visualizes price history, trading volume and daily price ranges
- Builds a Machine Learning model that predicts tomorrow's closing price with 99.42% accuracy

## 📊 Key Findings
- Reliance stock peaked at ₹3000+ in 2008 (Mukesh Ambani was world's richest man!)
- 2008 Global Financial Crisis caused a massive crash
- 2020-2021 saw highest ever trading volume due to Jio deals + COVID market crash
- Stock grew from ₹233 in 2000 to ₹2000+ in 2021

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn (Linear Regression)
- Streamlit

## 🚀 How to run locally
1. Clone this repository
2. Install requirements:
pip install pandas matplotlib scikit-learn streamlit

3. Run the app:
streamlit run app.py


## 📈 Model Performance
- Algorithm: Linear Regression
- Features used: Open, High, Low, Volume
- Accuracy: 99.42%
